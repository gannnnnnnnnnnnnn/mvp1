# Trial Feedback Template

Use these questions after a local trial run.

1. Was it clear how to get started from `/onboarding`?
2. Did the boundary setup make sense, or was any wording confusing?
3. Did Report feel trustworthy at a glance?
4. Did the Offset Mode explanation help you understand what was included or excluded?
5. Was Inbox useful for resolving things you were unsure about?
6. Did any page feel cluttered, too technical, or hard to navigate?
7. What part of the flow felt most valuable?
8. If this worked reliably on your own data, what would you expect to pay for it?
